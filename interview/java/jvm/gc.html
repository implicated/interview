垃圾收集器

Java 8 版本的 Hotspot JVM，默认情况下使用的是并行垃圾收集器（Parallel GC）。其他厂商提供的 JDK 8 基本上也默认使用并行垃圾收集器。

Java 9 之后，官方 JDK 默认使用的垃圾收集器是 G1。
